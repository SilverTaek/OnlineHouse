# Happyhouse
 코로나19 펜데믹으로 인한 부동산중개소와 고객과의 단절을 예방하기위해 비대면 서비스를 제공합니다.
 공공데이터를 활용하여 아파트,주택,코로나 선별진료소, 안심병원 등 고객에게 신뢰있는 검색 서비스를 제공합니다.
 회원과 중개사 모두 손쉬운 회원가입, 수정 여러 다양한 커뮤니케이션을 이용할 수 있습니다.
